## 자바 코드의 품질을 높이는 100가지 방법

> 타기르 발레예프(Tagir Valeev) 지음 · 정병열 옮김 · 한빛미디어
> 원서: *100 Java Mistakes and How to Avoid Them* (Manning)

자바에서 반복적으로 발생하는 **100가지 실수**를 주제별로 모아, 왜 문제인지와 어떻게 피하는지를 정리한 책입니다. 저자는 JetBrains의 기술 리드이자 Java Champion으로, IntelliJ IDEA 정적 분석기의 수많은 코드 인스펙션을 설계한 사람입니다. 그래서 "정적 분석기가 잡아내는 실수"의 시각으로 흔한 버그 패턴을 다룹니다.

### 정리 방식

- 챕터별로 폴더를 나눕니다.
- 각 챕터 폴더 안에는 관련 있는 실수 2~4개를 묶어 노트(`.md`)로 정리합니다.
- 문제 코드(안티패턴) → 왜 문제인가 → 해결 코드 → 메모 순서로 적습니다.
- 정리를 마친 항목은 아래 인덱스에서 체크하고 링크를 연결합니다.
